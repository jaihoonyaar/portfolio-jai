Projects

Merchant Cashback System

This is a fintech cashback system tailored for merchants who accept UPI payments. It allows them to define and automate their own reward schemes — like flat-percentage cashback, tiered milestones, or lottery-based bonuses — without needing integration with UPI apps like PhonePe or Google Pay. The backend listens to payment webhooks and instantly triggers payouts via APIs like RazorpayX or Cashfree. Featuring a React.js dashboard and a Node.js backend with PostgreSQL, the system provides real-time tracking of customer transactions, cashback disbursement, and ROI analytics — empowering merchants to boost customer retention through smart incentives.

Shark Tank

SharkTank is a full-stack web platform designed to simulate a real-world startup pitching environment for students. It enables young entrepreneurs to upload their startup ideas, manage pitch details, and receive structured feedback from mentors and judges. Built using React.js for the frontend and Node.js with MongoDB for the backend, it features role-based login, pitch scoring, and admin controls. The platform was successfully deployed during a campus hackathon, handling dozens of entries and fostering innovation, presentation skills, and entrepreneurial thinking among students.

EcoCycle

EcoCycle is an AI-powered web application that helps users identify and properly dispose of recyclable waste using real-time computer vision. By uploading or capturing an image of a waste item, the system detects the material type (plastic, metal, paper, etc.) and provides personalized recycling instructions, including local disposal center suggestions. Built with React.js for the frontend and powered by TensorFlow.js and Node.js on the backend, EcoCycle promotes environmental awareness through an intuitive, educational interface. The platform aims to make sustainability accessible and engaging for individuals and communities alike.
